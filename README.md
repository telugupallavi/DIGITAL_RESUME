## Features
- **Sticky Header Navigation:** Smooth scrolling and always visible on top  
- **Hero Section:** Animated background blobs with text reveal animations  
- **About Section:** Personal information, current role, and areas of interest  
- **Skills Section:** Display of technical skills using interactive cards  
- **Projects Section:** Showcasing selected projects with tech stack and descriptions  
- **Gallery Section:** Interactive image gallery with hover zoom effects  
- **Contact Section:** Email, LinkedIn, and downloadable resume  
- **Responsive Design:** Fully functional on desktop and mobile  
- **Animations:** Scroll-triggered fade, zoom, and hover effects

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript  
- **Frameworks & Libraries:** TailwindCSS, AOS (Animate on Scroll)  
- **Tools:** VS Code, Git, GitHub, DiceBear (for avatar)
